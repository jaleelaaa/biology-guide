How to Stop Server:
-------------------
1. Find the process:
   Windows: netstat -ano | findstr :8080
   Linux/Mac: lsof -ti:8080

2. Kill the process:
   Windows: taskkill /PID <process_id> /F
   Linux/Mac: kill <process_id>

Troubleshooting:
----------------
- If you see "Address already in use":
  Port 8080 is occupied. Stop other servers or use different port.

- If page doesn't load:
  Wait a few seconds and refresh. Check server.log for errors.

- If styles don't load:
  Check browser console (F12) for path errors.
